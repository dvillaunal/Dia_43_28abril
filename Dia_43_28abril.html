Qu son los bucles? Definicin:

Un bucle es una instruccin de programacin que se repite hasta que se alcanza una condicin especfica.

El bucle ejecuta un bloque de cdigo una y otra vez hasta que no se requiera ninguna otra accin.

Cada vez que se ejecuta el bloque de cdigo dentro del bucle se llama una iteracin.

Diferentes tipos de bucles

Dependiendo de su situacin de programacin especfica, puede necesitar diferentes estructuras de bucle que ejecuten los bloques de cdigo dentro del bucle en base a diferentes condiciones.

en R generalmente proporciona tres tipos diferentes de bucles: for-loops, while-loops y repeat-loops. El siguiente grfico ilustra el flujo de trabajo de cada uno de los tres tipos de bucles:

Ciclo for:

La estructura for nos permite ejecutar un bucle (loop), realizando una operacin para cada elemento de un conjunto de datos.

Sintaxis:

Su estructura es la siguiente:

for(elemento in objeto){

operacion_con_elemento

}

Con lo anterior le decimos a R:

  • PARA cada elemento EN un objeto, haz la siguiente operacin.

Al escribir un bucle for la parte que corresponde al elemento la podemos llamar como nosotros deseemos, pero la parte que corresponde al objeto debe ser el nombre de un objeto existente.

# Definimos un vector pseudo-aleatorio:

Nota:

Notars que al ejecutar el cdigo anterior parece que no ha ocurrido nada. En realidad, s se han realizado las operaciones, pero R no ha devuelto sus resultados.

Las operaciones en un for se realizan pero sus resultados nunca son devueltos automticamente, es necesario pedirlos de manera explcita.

A diferencia de otros lenguajes de programacin en los que pedimos los resultados de un bucle con return(), en R este procedimiento slo funciona con funciones.

While

Este es un tipo de bucle que ocurre mientras una condicin es verdadera (TRUE). La operacin se realiza hasta que se se llega a cumplir un criterio previamente establecido.

Sintaxis

while(condicin){

  operaciones

}

Con esto le decimos a R:

  • MIENTRAS esta condicin sea VERDADERA, haz estas operaciones.

La condicin generalmente es expresada como el resultado de una o varias operaciones de comparacin, pero tambin puede ser el resultado de una funcin.

Ejemplos:

Vamos hacer correr el programa, hasta que los valores pseudo-aleatorios se parezcan, (se crece de dos en dos):

repeat

Este es un bucle que se llevar a cabo el nmero de veces que especifiquemos, usando un break para detenerse. repeat asegura que las operaciones que contiene sean iteradas al menos en una ocasin.

Los ciclos de repeticin siguen una lgica similar a los ciclos while, ya que tambin se pueden usar cuando el usuario no sabe el nmero exacto de veces que se debe repetir el cdigo R. Sin embargo, los bucles de repeticin no son tan populares como los bucles while.

Sintaxis:

La estructura de repeat es el siguiente:

`repeat` {
  
  operaciones
  
  _un break para detenerse_
}

Nota:

  1. Si no incluimos un break, el bucle se repetir indefinidamente y slo lo podremos detener pulsando la tecla ESC, as que hay que tener cuidado al usar esta estructura de control.

  2. Este tipo de bucle es quizs el menos utilizado de todos, pues en R existen alternativas para obtener los mismos resultados de manera ms sencilla y sin el riesgo de crear un bucle infinito. Sin embargo, puede ser la mejor alternativa para problemas especficos.

# Vector con un valor => 0:
message("\n# Vector con un valor => 0:")
## 
## # Vector con un valor => 0:
x.re <- 0

# Ahora creamos nuestro metodo de control repeat:
message("\n# Ahora creamos nuestro metodo de control repeat")
## 
## # Ahora creamos nuestro metodo de control repeat
repeat {
  x.re <- x.re + 1
  print(x.re)
  
  if (x.re >= 5){
    break
  }
}
